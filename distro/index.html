Carrusel responsivo

Una alternativa para mostrar una galería de imágenes responsiva.

Esto no es propiamente una aplicación de un carrusel, sino un script que ajuste la imagen al tamaño de la pantalla, es decir, proporcional al viewport o como se diría en el mundo del HTML, responsivo.
